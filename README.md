# Client-Server

HTTP и HTTPS
   ```
   Http - hypertext transfer protocol
   клиент-сервеный протокол
   
   HTTP запрос состоит из:
    -стартовой строки (URL, метод запроса)
    -заголовков (пр. инф-я о браузере, языке, авторизации)
    -тела сообщения (пр. комментарий к видео)
    ```
    
    ```
    Https - http + secure
    
    Шифрование обеспечивают механизмы SSl и TLS
    
    SSL (secure sokets layer) работает так:
    когда мы подключаемся к сайту, просим у него SSL сертификат, браузер проверяет его, если все ок, то начинается обмен шифрованными данными
    
    TLS (transport layer security) - прокачанная версия SSL. По сути, последняя версия SSL - это TLS 1.0, но сейчас применяются версии TLS 1.2 и 1.3
    ```
